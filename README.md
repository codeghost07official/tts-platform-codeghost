# VoiceTTS 🎙️

VoiceTTS is a fast, minimal and powerful Text-to-Speech (TTS) web tool that converts text into natural-sounding voice instantly — **for free**.

No login. No limits. No unnecessary complexity.

---

## 🚀 Features

* 🔊 Convert text to speech instantly
* 🎧 Listen to audio directly in browser
* ⬇️ Download generated audio
* 🎙️ Multiple high-quality voices (Male & Female)
* 🌗 Dark / Light theme toggle
* ⚡ Clean, fast, and minimal UI
* ❌ No signup required — **completely free**

---

## 🛠️ Tech Stack

**Frontend**

* HTML
* CSS
* JavaScript

**Backend**

* Python
* Flask
* edge-tts

---

## ⚙️ How It Works

1. User enters text in the input box
2. Selects a voice
3. Clicks **Listen** or **Download**
4. Backend processes the text using TTS engine
5. Audio is generated and returned instantly

## ▶️ Running Locally

### 1. Clone the repository

```
git clone https://github.com/YOUR_USERNAME/tts-platform-codeghost.git
cd tts-platform-codeghost
```

### 2. Setup backend

```
cd backend
python3 -m venv venv
source venv/bin/activate
pip install flask flask-cors edge-tts
```

### 3. Run backend

```
python app.py
```

### 4. Open frontend

Open `frontend/index.html` in your browser.
